# AGENTS.md - BalaBharati

This repo hosts a simple static Q&A game for kids on GitHub Pages.

## Purpose

- keep the site light, colorful, and easy to run
- prefer plain HTML, CSS, and JavaScript
- avoid frameworks unless there is a clear need

## Rules

- keep the site static so GitHub Pages works without a build step
- keep text kid-friendly and easy to scan
- prefer large buttons, clear colors, and obvious feedback
- keep docs short
- do not add secrets or tokens

## Repo structure

- `README.md`: quick overview
- `AGENTS.md`: repo-specific rules
- `AGENTS.override.md`: local/private override
- `PLANS.md`: active work
- `docs/ROADMAP.md`: next steps
- root HTML/CSS/JS files: site runtime

## Current focus

- publish the first playable GitHub Pages version
- keep the repo simple enough for easy edits later
